Projeto pronto para Vercel / Netlify (Vite + React + Tailwind)

Como usar (local):
1. Extraia o ZIP
2. No terminal, entre na pasta do projeto:
   cd tela-silk-arte-site
3. Instale dependências:
   npm install
4. Rode em desenvolvimento:
   npm run dev
5. Abra no navegador o endereço mostrado pelo Vite (ex: http://localhost:5173)

Deploy no Vercel:
1. Crie conta em vercel.com
2. Faça 'Import Project' -> selecione o repositório (ou arraste o ZIP)
3. Vercel instala dependências automaticamente e publica o site.

Observações:
- Substitua as imagens mockup em src/ por suas fotos reais.
- Substitua o logo e gere um QR code para o link do WhatsApp.
- Se quiser, eu posso gerar as imagens mockup e a logo adaptada.
